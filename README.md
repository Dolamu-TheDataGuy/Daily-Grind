# 🧠 Daily Grind — Problem Solving Log

A daily log of coding problems I solve across **Go, Python, C, and SQL**, with technical write-ups covering how each solution works, the challenges I hit, eureka moments, and lessons learnt.

> The goal isn't just solving problems — it's *understanding* them well enough to explain them.

## Why this repo exists

- 📈 **Consistency** — one problem a day, documented, no exceptions.
- ✍️ **Understanding** — if I can't write up how it works, I don't understand it yet.
- 🐛 **Debugging skill** — the write-ups capture what broke and why, not just what worked.
- 🗂️ **Portfolio** — a verifiable, growing record of problem-solving across languages.

## Structure

Each problem lives in its own folder by language and date:

```
daily-grind/
├── go/
│   └── 2026-06-12-concurrent-task-runner/
│       ├── solution.go
│       └── README.md      ← the write-up
├── python/
├── c/
├── sql/
└── TEMPLATE.md            ← write-up template for every entry
```
